Margin, Border & Padding in CSS

April 16, 2021 by Ryan

This blog post explores the difference between the margin, border and padding properties of elements in CSS as well as the Box Model.

Margin, Border & Padding are all CSS properties, that change how elements are positioned and interact with others on a webpage. The three properties behave in similar ways and are often confused, however they function in distinct ways. Margin, Border & Padding combined are referred to as the "CSS Box Model", or Box model for short.

Margin

Margins clear an area outside the border and are always transparent. Margins can be thought of as the space between the border of one element and the next.

Four margins need to be declared if they are not to collapse onto each other:

  • margin-top: x;
  • margin-right: x;
  • margin-bottom: x;
  • margin-left: x;

Margin properties are usually defined in a clockwise order, starting from the top and moving around the element.
Margins can also be defined with shorthand. This condenses the 4 margin properties onto a single line, which is written as:

  • margin: x x x x;

The 4 values following the margin follow the clockwise order of margins.
