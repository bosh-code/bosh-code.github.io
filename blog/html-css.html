What's the difference between Margin, Border & Padding?

Margin

 The margin is the amount of space outside an element. It is a fixed size and can be changed through the use of CSS. The margin is the distance between and element and another.

Border

 A border is like a photo frame. It encloses the outside of the element and any padding.

Padding

 Padding is a defined space between an element and it's border or bounds. If no border is applied, it is the space between an element and it's parent, less the margin.

The Box Model

 The box model is a visual representation of how margins, borders and padding are added to an element.
