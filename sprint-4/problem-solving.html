Getting blocked

I've written the Fizzbuzz code a few times, it's usually what I use after writing Hello world for each language. Everytime there is a roadblock, as languages are different. I know the logic well, however language syntax and environments can trip me up early on. With Fizzbuzz, I wanted to update a printout of the result of running the code on my HTML page. That's simple DOM manipulation and something I had just been practicing, so I felt confident.

However, after initially testing it, I found that my result printout was being appended to the existing text in the element. I rubber ducked the javascript I had wrote to isolate where I believed the issue was, then looked for a solution. I was able to use terms and keywords I had learnt to effectively search StackOverflow for an answer and example, that I as able to re-write, changing it to fit my use case.

The code that solved my problem is this:

function fizzbuzz (num) {
  document.getElementById('result').innerHTML = '' // clear the paragraph first
...
function append (textToAppend) {
  let paragraph = document.getElementById('result') // get the paragraph
  let text = document.createTextNode('// ' + textToAppend) // get the text to append to the paragraph
  let linebreak = document.createElement('br') // create a linebreak to append
  paragraph.appendChild(text) // append text
  paragraph.appendChild(linebreak) // append linebreak
}
