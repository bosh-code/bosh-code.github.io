Control flow graphs

An example control flow graph about making press coffee

In this example control flow graph we can multiple examples of control flow logic. For example, if there is enough coffee, we continue, else we grind more coffee. We can write this in pseudocode as:

if (coffeeAmount < amountRequired) {
 grind more
} else { 
 continue
}
If we look further into the "grind more" block, we can see this can be written as a while loop:
if (coffeeAmount < amountRequired) {
 while (coffeeAmount < amountRequired) {
  grindMore();
  coffeeAmount++;
 }
}
